About

Purpose

There are a couple of reasons I decided to make this website. Below are a few:

Learning web hosting and server stuff

Sometime around the fall of 2019, I decided to try out Linux (eventually settling on the Manjaro distribution). I grew to really like it for its customizability, programmer-friendliness, and command line. As a result, I became increasingly interested in running Linux in a server setting, both as a challenge to myself and to learn more about it. Ultimately, I decided that setting up a web server would be a pretty cool project, so that's what I decided to do.

Learning front-end web development

Prior to making this website, I did not know much about front-end web development. Building this website has not only provided me with experience in doing so, but it has also opened my eyes to the different ways that one can build a website and the various things one must take into consideration when doing so. I will likely continue using this site to display the results of my front-end development experiments.

Making blog posts

The final reason I decided to make a website is to publish written tutorials/guides about software or other topics that interest me. This would include things like material for a Linux club I am thinking about starting at my school next year, or a written complement to a YouTube video.

How I hosted this website

This website is hosted entirely on Google Cloud's free tier, on an f1-micro VM instance in Iowa. I used to use Debian as the VM's operating system but I later switched to Google's Container-Optimized OS since I found Debian a little too bloated. The only disadvantage is that there is no package manager in Container-Optimized OS so everything needs to be run in docker containers. What the server is running is loosely based off of this image I made, thus this website runs on NGINX. The SSL certificates (for HTTPS) are taken care of by the EFF's Certbot.

How I designed this website

This website was designed using vanilla HTML, CSS, and JavaScript. The font used is Fira Code, and most of the icons were found on Ionicons and modified to work as HTML <svg> elements. The GNU head on the home page was found on the GNU website, created by Aurelio A. Heckert <aurium@gmail.com>, and released under the Free Art License.
